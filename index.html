PocketPilot: The Personal Expense Compass

Java JSP/Servlets MySQL HTML/CSS/JS Machine Learning

Overview

PocketPilot is a web-based financial tracking application designed to help users navigate their daily transactions with clarity. The project features automated budgeting, category distribution tracking, and responsive charts showing spending behaviors over time.

Key Features

  • AI/Automation: Integrates custom algorithms and simple machine learning models to automatically categorize user transactions based on history and payment descriptions.
  • Habit Analysis: Identifies potential wasteful spending sequences and gives insights on how to stay within monthly budgets.
  • Data Integrity: Utilizes transactional SQLite/MySQL database systems with structured Java Servlets managing sessions, secure auth, and concurrent operations.

DIY Wind Speed Monitoring System

MicroPython Raspberry Pi Pico W ThingSpeak Cloud Climate Data

Overview

Built as part of an IoT Computing module, this project consists of designing and calibrating a custom anemometer connected to a Raspberry Pi Pico W micro-controller. Telemetry logs are broadcast over local Wi-Fi to a remote cloud interface.

Key Features

  • Sensor Calibration: Wrote custom MicroPython firmware measuring analog pulse intervals from anemometer rotations, converting frequency values into physical wind speed (m/s & knots).
  • Real-time Telemetry: Configured Wi-Fi networking routines on the Pico W to push hourly readings securely to the ThingSpeak Cloud dashboard.
  • Marine Application: Designed with coastal wind profiles in mind, illustrating practical data visualization for maritime safety and harbor operations.
